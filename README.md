# JV Driver - App Paquetería

Proyecto Android nativo completo para generar APK con GitHub Actions.

## Cómo generar la APK
1. Sube todos estos archivos a un repositorio GitHub.
2. Entra en Actions.
3. Ejecuta Build APK.
4. Descarga el artifact app-debug-apk.
5. Dentro estará app-debug.apk.

## Demo incluida
- Login conductor.
- Ruta diaria demo.
- Detalle de entrega.
- Firma en pantalla.
- Estado entregado/ausente.
- Generación de PDF local.
